# Welcome

## What is BunyJS

BunyJS is Ioc framework not just library, and it is designed to be simple and easy to use and to be able to create complex applications.

## Why BunyJS

BunyJS provide many packages to help you to create your application, and the simplicity to add other packages easily without any conflict.

## Features

BunyJS comes with a number of features that make it a powerful tool for building complex applications:

- **Powerful IoC container:** BunyJS has a powerful Inversion of Control (IoC) container that helps manage your classes, interfaces, and other types.

- **Smarter dependency injection:** BunyJS provides a smarter way to handle dependency injection, making it easier to manage dependencies in your application.

- **Fast compiler:** BunyJS comes with a fast compiler that helps you compile your application quickly and efficiently.

- **Easy to use:** BunyJS is designed to be easy to use, with a simple and intuitive API that makes it easy to get started.

- **Modular:** BunyJS is designed to be modular, with a number of packages that you can use to extend its functionality.
